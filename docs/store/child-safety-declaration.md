# Play Console「児童の安全に関する基準の宣言」入力メモ（CSAE）

Play Console → アプリのコンテンツ → **児童の安全に関する基準の宣言**（Child safety standards）で入力する内容の下書き。

## 該当理由
Nostrism は **ソーシャル / ユーザー生成コンテンツ(UGC) / 対人コミュニケーション（DM）** を含むため、児童の安全に関する基準（CSAE）の宣言が必要。

## 入力内容
| 項目 | 値 |
|---|---|
| **公開された児童安全基準のURL** | `https://nostrism.shino3.net/child-safety.html` |
| **児童の安全に関する連絡先メール** | `<記入>`（#29 で保留中） |
| **CSAE 関連法令の遵守** | はい |
| **アプリ内の通報手段はあるか** | **はい** — 各投稿の ⋯ メニュー →「通報」で NIP-56（kind:1984）レポートを発行。児童の安全に関わる内容は「違法」を選択。ミュート/ブロックも可 |
| **CSAM を確認した場合に当局へ通報するか** | はい（法令に従い対応。利用者にも NCMEC 等への通報を案内） |

## 補足
- コンテンツは分散型 Nostr のリレー（第三者）に保存されるため、ネットワークからの完全削除にはリレー運営者の対応が必要。アプリ側でできる措置（通報の発行・表示制限・ミュート）を提供。
- 実装: 通報 = `EventRepository.reportNote()`（NIP-56 kind:1984）。UI = NoteItem の ⋯ →「通報」→ 理由ピッカー。
